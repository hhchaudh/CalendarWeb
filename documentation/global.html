addTab((f,l))

method to generate month specific table

Parameters:
Name Type Description
(f,l) 'f' is the day of the week, i.e. Sunday, Monday, that the month begins on. 'l' is the number of days in that month. A table is then dynamically generated by starting to append cells from the 'f' on row 0, to 'l' how many rows are needed to complete the month.
